# Sora Farewell Studio — Prompt-to-Storyboard Sandbox

A single-file web app inspired by the trend **"Goodbye to Sora"**.

Instead of generating video pixels, it generates the structure you need to make video generation (or human filming) work well:
- Shot-by-shot storyboard generator (deterministic, local)
- Procedural kinetic preview that reacts to shot metadata
- Timeline view with quick edit/preview buttons
- Shot editor (title, prompt, duration, camera) + deterministic "auto-polish"
- Reorder / duplicate / delete shots
- Export storyboard to local JSON
- Shareable link that encodes state in the URL hash (no server)

Open `index.html` in any modern browser.
